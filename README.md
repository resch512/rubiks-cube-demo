# Rubik's Cube Demo

Simple mobile-friendly browser demo showing a 3D Rubik's Cube.

## Features

- Starts solved
- **Mix** button: applies a random scramble with animation
- **Solve** button: solves the cube by applying the inverse algorithm with animation
- Animation speed is ~0.5s per turn
- Works in mobile browsers

## Run locally

Open `index.html` in a browser.

## Deploy

Static hosting friendly (GitHub Pages).
